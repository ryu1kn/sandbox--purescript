
# PureScript Spike

## Prerequisite

* purescript. `npm install -g purescript`
* pulp (build tool). `npm install -g pulp`
* bower (package manager). `npm install -g bower`

## Test

```sh
$ pulp test
```

## Run the programme

```sh
$ pulp run
```

## REPL

```sh
$ pulp repl
```

## Introducing new libraries

Use bower. e.g.

```sh
$ bower install purescript-lists --save
```
